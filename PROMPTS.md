# AI Prompt Record

This repository keeps the source prompts used for its sprint-based AI-assisted development in the [`prompts/`](prompts) directory.

## Organization

The prompt files are retained as project history and are named sequentially from `Sprint_01.md` through `Sprint_20.md`. They cover project setup, authentication, user management, vehicle management, purchase and sales workflows, dashboard implementation, and final release review.

## Use of prompts

The prompts define sprint objectives, architecture constraints, acceptance criteria, and requested verification. They were used to guide implementation and review; they are not a substitute for developer validation. The project README's **My AI Usage** section describes the developer's review and accountability responsibilities.

## Current record

The latest finalization prompt is [`prompts/Sprint_20.md`](prompts/Sprint_20.md). No conversations have been reproduced or fabricated in this file; the original sprint prompt files remain the authoritative record.
